# Project Summary - AI Voice Agent

## Overview

A real-time conversational AI voice agent built for Techfest 2025-26 Murf Hackathon. Demonstrates natural voice synthesis using Murf Falcon API with advanced customization features.

## Key Features

### Core Functionality
✅ Real-time speech recognition (Web Speech API)
✅ Natural voice synthesis (Murf Falcon API)
✅ Two-way conversational interaction
✅ Automatic speech detection and processing

### Customization Features
✅ 6 speaking styles (Conversational, Professional, Friendly, Casual, Formal, Energetic)
✅ Speech rate control (-50 to +50)
✅ Pitch adjustment (-50 to +50)
✅ Real-time settings application

### User Experience
✅ Modern, professional UI (React + Tailwind CSS)
✅ Real-time transcript display
