# Demo Video Script

## Duration: 2-3 minutes

### Introduction (15 seconds)
"Hi! I'm [Your Name], and this is my AI Voice Agent built for the Techfest 2025-26 Murf Hackathon. This application demonstrates real-time conversational AI using Murf Falcon's text-to-speech API."

### Feature Showcase (90 seconds)

**1. Basic Conversation (20 seconds)**
- Show the interface
- Click "Start Recording"
- Say: "Hello, what can you do?"
- Show the agent responding with voice
- Highlight the transcript appearing in real-time

**2. Voice Customization (30 seconds)**
- Show the voice dropdown
- Switch between different voices (Ken, Natalie, Sarah)
- Demonstrate the difference in voices
- Say: "Tell me about Murf Falcon"

**3. Advanced Settings (20 seconds)**
- Open Advanced Settings panel
- Adjust speech rate slider
- Adjust pitch slider
- Demonstrate the difference with a test phrase
- Say: "What time is it?"

**4. Speaking Styles (20 seconds)**
- Change speaking style dropdown
- Try Professional, Conversational, Friendly
- Show how the same text sounds different
- Say: "This is a test of different speaking styles"

### Technical Highlights (30 seconds)
"The backend is built with Python and Flask, using the Murf Falcon API for text-to-speech generation. The frontend uses React and Tailwind CSS for a modern, responsive interface. Speech recognition is handled by the Web Speech API for real-time input."

### Key Features Summary (15 seconds)
"Key features include:
- Real-time speech recognition
- Multiple voice options
- Customizable speech rate and pitch
- Different speaking styles
- Secure API key management"

### Closing (15 seconds)
"This project is built using Murf Falcon – the consistently fastest TTS API. The response time is incredibly fast, making conversations feel natural. Check out the code on GitHub, and thanks for watching!"

## Recording Tips

1. **Screen Recording**: Use OBS Studio or similar
2. **Audio**: Use a good microphone for clear narration
3. **Resolution**: Record in 1080p
4. **Editing**: Keep it concise, cut any dead time
5. **Background**: Clean, professional workspace
6. **Lighting**: Good lighting for webcam if showing yourself
7. **Practice**: Do a few test runs before final recording

## What to Show

✅ Clean, professional UI

✅ Smooth, responsive performance
